This project uses Python language & DJango framework. 
If need install DJango, use "pip install django" from the command line in PyCharm.
Run the "manage.py" to run the program. 
When the server starts up, it will give you an error in the web browser; to fix this, just put "products" at the end of the url, so it will look something like: "http://127.0.0.1:8000/products".

Funtions Including:
Main page that shows products; 
Add product page; 
The button to add items to cart; 
Search items; 
Adding items to cart and specifying quantity; 
View the shopping cart; 
Calculating and displaying the total price; 
If click "Place order" in the shopping cart screen, the cart is cleared and a message is displayed.
